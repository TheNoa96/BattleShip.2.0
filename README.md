# BattleShip2.0
Famoso juego de estrategia

El juego consiste en que posiciones tu flota en el tablero y despúes trates de hundir la flota enemiga adivinando la posición.


## Para Instalar:
Npm Install

## Para Correr:
Npm Start

## Tecnologias utilizadas:
- React
- Bootstrap
- Javascript
- Html
- Css
- Styled-components
